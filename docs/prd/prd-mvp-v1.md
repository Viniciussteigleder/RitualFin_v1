# PRD RitualFin MVP v1

- PT-BR web app, EUR currency.
- Login (Google + Email/Password) via Supabase.
- Upload 1 Miles & More CSV, parse/enrich/dedupe/transacoes.
- Rules engine keyword matching on `desc_norm`.
- Confirm queue resolves conflicts and creates rules.
- Dashboard mensal com spent, receitas, budget, remaining e projeção.
